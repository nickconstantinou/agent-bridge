## Non-gaps (working architecture to preserve — do not replace)
- Handler-map dispatch in jobExecutor (extend into workflow engine, keep the seam).
- Disposable git workspaces (`workspace.ts`) — matches agent-orchestrator worktree isolation intent already.
- Platform seam (`platform.ts`) for Telegram/Discord.
- EventStore append path — correct foundation for G4, extend rather than rebuild.
- Skill packs — become the library the workflow engine references.
- Circuit breaker, orphan cleanup, KillMode=control-group, session TTL — keep verbatim.
